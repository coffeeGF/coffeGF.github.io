博客搭建

发表于 2017-11-08
目的：瞎折腾
借鉴：dimsky大神的简书传送门

一、需准备的内容

1.github账号，并创建repository
2.通过nodejs安装hexo博客框架

二、创建repository

注意userName的配置，其他省略，见上面教程。

三、安装nodejs

传送门中的教程，是使用nodejs管理工具nvm。实际安装过程中出现各种command not found情况。
遂，弃。直接 

$ brew install node

一样好用。

四、安装hexo

通过nodejs提供的npm工具安装hexo

$ sudo npm install hexo

安装自动部署发布工具

$ npm install hexo-deployer-git --save

五、发布博客

1.初始化博客文件 

$ hexo init username.github.io

2.配置博客

$ cd username.github.io
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

3.编写博客

$ cd username.github.io/source/_posts
$ vi first.md

4.预览
执行成功之后，可以在https://localhost:4000预览

$ hexo s

5.发布

$ hexo clean && hexo g && hexo d

五、博客地址

http://userName.github.io
